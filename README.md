# 🚀 Smart Task Manager

A modern multi-page productivity web application built using **HTML, CSS, and JavaScript** with localStorage support.

---

## 📌 Features

- 🔐 User login (stored in localStorage)
- 📝 Add / Delete / Complete tasks
- 🎯 Goals tracking system
- ⏰ Reminders section
- 📊 Live dashboard statistics
- 🎉 Confetti animation on task completion
- 💾 Data saved in browser (no backend required)
- 📱 Clean responsive UI with sidebar navigation

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage API

---

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html`
3. Start using the app in your browser
